# Training on UniACT Dataset

## Data Preparation

We support the following open-source datasets—pick whichever best fits your needs:

- OXE: [Converted LeRobot v2 Data](https://huggingface.co/collections/IPEC-COMMUNITY/openx-LeRobot)
- OXE-AugE: [LeRobot v3 Dataset Source](https://huggingface.co/oxe-auge/datasets)
- RoboCoin: [LeRobot v2 Dataset Source](https://modelscope.cn/organization/RoboCOIN?tab=dataset)
- RoboMind: [Source Data](https://huggingface.co/datasets/x-humanoid-robomind/RoboMIND)
- AgibotWorld-Beta: [Our Converted LeRobot v2 Data](https://modelscope.cn/organization/RoboCOIN?tab=dataset)
- InternData: [LeRobot v2 Dataset Source in Simulation](https://www.modelscope.cn/datasets/InternRobotics/InternData-A1/tree/master/sim_updated)
- Galaxea: [LeRobot v2 Dataset Source](https://modelscope.cn/datasets/Galaxea/Galaxea-Open-World-Dataset/tree/master/LeRobot)

#### 1. Download the dataset

You can download the dataset from the links above, or from any other source if that is more convenient.

Our training code mainly targets LeRobot V2, so we recommend downloading or converting your data to LeRobot V2 format.

#### 2. Convert the dataset to LeRobot V2

We build on [Any2LeRobot](https://github.com/Tavish9/any4lerobot/tree/main); the related code is under `data_process/any4lerobot`.
The upstream [LeRobot](https://github.com/huggingface/lerobot) repository is also provided under `data_process/lerobot`.
Prepare a conda environment following the [installation instructions](https://github.com/amap-cvlab/ABot-Manipulation/tree/main/data_process/lerobot#installation).
These tools can convert AgibotWorld-Beta and RoboMind to LeRobot V3 format.
LeRobot V3 data can be further converted to LeRobot V2.1.

#### 3. Update the `/meta` files

The `/meta` directory is an important part of a LeRobot dataset.
We compute and update meta files for our pretraining setup.
Download the preprocessed `/meta` files from [Abot-M0-MetaData](https://www.modelscope.cn/datasets/amap_cvlab/Abot-M0-MetaData), extract it, and place each dataset’s `meta` folder next to the matching LeRobot V2 dataset on disk.

##### Details of the downloaded `/meta` files:

- `episodes.jsonl`: (Original LeRobot file) Episode-level index; one line per episode with basic metadata.
- `episodes_stats.jsonl`: (Original LeRobot file) Episode-level statistics (e.g., length and simple aggregates).
- `info.json`: (Original LeRobot file) Dataset-level metadata (version, schema, high-level info).
- `modality.json`: (New, ensure it exists) Defines modality keys and layout for the dataloader.
- `stats_delta_state.json`: (New, ensure it exists) Statistics for delta-state normalization in training/inference.
- `stats_gr00t.json`: (Original LeRobot file) Extra statistics for GR00T-style preprocessing.
- `steps_data_index_train_test_ratio_0_0001_shuffle_seed_42.pkl`: (New, auto-generated by our code) Cached step index for the train split (fixed ratio and shuffle seed).
- `steps_data_index_test_test_ratio_0_0001_shuffle_seed_42.pkl`: (New, auto-generated by our code) Cached step index for the test split (same ratio and seed).
- `tasks.jsonl`: (Original LeRobot file) Task-level annotations linking episodes or steps to task semantics.

If you want to compute `stats_delta_state.json` yourself, you can run the provided script:

```bash
cd examples/Pretrain
bash run_compute_delta_state_stats.sh
```

#### 4. Add your dataset path to `.yaml` config file and `mixtures.py`

In config file (such as `examples/Pretrain/ABot_pretrain.yaml`):

```yaml
datasets:
  vla_data:
    dataset_py: LeRobot_datasets
    data_root_dir:  # path to your dataset
    data_mix: pretrain
```

In `dataloader/gr00t_lerobot/mixtures.py`:

```python
DATASETS_ROOT = Path("")  # path to your dataset
```

---

## Check Your Dataloader

We provide a simple way to check your dataloader. Make sure you can load batched data successfully:

```bash
python ABot/dataloader/lerobot_datasets.py --config_yaml examples/Pretrain/ABot_pretrain.yaml
```

---

## Framework Preparation

Before training, ensure that your framework can run both `forward` and `predict_action` with a fake data example.

Try the following command:

```bash
python ABot/model/framework/ABot_M0.py --config_yaml examples/Pretrain/ABot_pretrain.yaml
```

---

## Training

Once everything is ready, use the provided script to start training:

```bash
bash ./examples/Pretrain/run_pretrain.sh
```

---

## Note

⚠️ Ensure that the script explicitly uses the validated config path in `run_LeRobot_datasets.sh`.
If it is not already passed, add the `--config_yaml` argument.
